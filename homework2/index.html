Laboratorio Interattivo: Dataset e Distribuzioni

Un viaggio interattivo tra dati, statistica e crittografia.

1. Cos’è un Dataset

Un dataset è una collezione strutturata di dati. Ogni riga rappresenta un’osservazione, mentre ogni colonna rappresenta una variabile. I dataset consentono di condurre analisi, scoprire relazioni e prevedere fenomeni.

2. Cos’è una Distribuzione

Una distribuzione mostra come si ripartiscono i valori di una variabile nel dataset. Le distribuzioni univariate riguardano una sola variabile, mentre le bivariate confrontano due variabili per trovare correlazioni o pattern.
